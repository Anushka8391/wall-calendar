# Interactive Wall Calendar

A premium interactive calendar component built using Next.js, Tailwind CSS, and modern UI practices.

## 🚀 Features
- Month navigation with smooth animations
- Date selection and range selection
- Notes panel for adding daily notes
- Dynamic background images per month
- Responsive and clean UI

## 🛠 Tech Stack
- Next.js 14
- Tailwind CSS
- TypeScript
- Framer Motion

- ## 💡 Highlights
- Built with scalable component architecture
- Optimized UI/UX with smooth animations
- Clean code structure for maintainability
